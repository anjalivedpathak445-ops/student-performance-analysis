# 🎓 Student Performance Analysis

## 📌 Overview

This project analyzes student performance data using Python.
It explores how different factors (such as study time, scores, etc.) affect student results.

---

## 🛠️ Technologies Used

* Python 🐍
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook
